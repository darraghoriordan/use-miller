Encode or decode HTML entities and Unicode escape sequences. Convert special characters for safe use in HTML or JavaScript.

![HTML Char Encoding](images/html_char.png)

## Features

- **Encode/Decode modes** - Convert in either direction
- **HTML Entity format** - Named entities like `&amp;`, `&lt;`, `&copy;`
- **Unicode format** - Escape sequences like `\u0026`, `\u003C`

## How to Use

1. Paste your text in the input field
2. Select "Encode" or "Decode"
3. Select the format: "Html Entity" or "Unicode"
4. Click "Submit"

## When to Use Each Format

- **HTML Entities** - For embedding special characters in HTML documents
- **Unicode Escapes** - For embedding special characters in JavaScript strings or JSON

## Examples

| Character | HTML Entity | Unicode  |
| --------- | ----------- | -------- |
| &         | `&amp;`     | `\u0026` |
| <         | `&lt;`      | `\u003C` |
| >         | `&gt;`      | `\u003E` |
| "         | `&quot;`    | `\u0022` |
