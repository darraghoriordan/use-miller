Convert text between 11 different case formats. Perfect for transforming variable names, file names, or text between different naming conventions.

![String Case Converter](images/stringcase.png)

## Supported Cases

| Case          | Example            |
| ------------- | ------------------ |
| camelCase     | `myVariableName`   |
| Capital Case  | `My Variable Name` |
| CONSTANT_CASE | `MY_VARIABLE_NAME` |
| Header-Case   | `My-Variable-Name` |
| kebab-case    | `my-variable-name` |
| lower case    | `my variable name` |
| PascalCase    | `MyVariableName`   |
| Sentence case | `My variable name` |
| snake_case    | `my_variable_name` |
| Title Case    | `My Variable Name` |
| UPPER CASE    | `MY VARIABLE NAME` |

## How to Use

1. Paste your text in the input field (each line is processed separately)
2. Select the target case from the dropdown
3. Click "Submit"
4. Copy the result

## Multiple Lines

New lines are treated as separators. Each line is converted independently, allowing you to batch-convert multiple strings at once.
