Configure Local Dev Tools settings including file paths, AI API keys, and feature options.

## File Paths

| Setting           | Description                       | Default         |
| ----------------- | --------------------------------- | --------------- |
| Git Projects Path | Directory to scan for git repos   | Home directory  |
| Global Git Config | Path to global git config         | `~/.gitconfig`  |
| SSH Config File   | Path to SSH config                | `~/.ssh/config` |
| Chrome History    | Path to Chrome history (optional) | Auto-detected   |

Use the folder/file picker buttons to browse for paths.

## AI Configuration

Required for Marketing Week and other AI features:

| Setting        | Description                |
| -------------- | -------------------------- |
| OpenAI API Key | Your OpenAI API key        |
| OpenAI Org ID  | Organization ID (optional) |
| Gemini API Key | Google Gemini API key      |
| LM Studio URL  | Local LM Studio server URL |

## Marketing Week Options

| Setting               | Description                             |
| --------------------- | --------------------------------------- |
| Enable Marketing Week | Show/hide the Marketing Week tool       |
| Persona               | Description of your voice and audience  |
| Twitter Handle        | Your Twitter username                   |
| LinkedIn URL          | Your LinkedIn profile URL               |
| GitHub PAT            | Personal access token for commit access |

## Setup Wizard

Click "Run Setup Wizard" to walk through initial configuration step by step. This is helpful for first-time setup or when reconfiguring the application.
