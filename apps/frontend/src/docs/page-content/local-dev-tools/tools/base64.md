Encode text to Base64 or decode Base64 strings back to plain text. Uses Node.js crypto for reliable encoding.

![Base64 Encode/Decode](images/base64.png)

## Features

- **Encode** - Convert plain text to Base64
- **Decode** - Convert Base64 back to plain text

## How to Use

1. Paste your text or Base64 string in the input field
2. Select "Encode" or "Decode"
3. Click "Submit"
4. Copy the result

## What is Base64?

Base64 is a binary-to-text encoding scheme that represents binary data as ASCII characters. It's commonly used for:

- Embedding images in HTML/CSS
- Encoding data in URLs
- Storing binary data in JSON
- Email attachments (MIME)
