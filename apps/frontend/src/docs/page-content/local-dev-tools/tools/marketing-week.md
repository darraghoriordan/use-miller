AI-powered content generation tool for developers. Generate social media content ideas from your git commits for Twitter, LinkedIn, and blog posts.

## API Notice

This tool uses AI services and your data will be sent to the selected provider. You use your own API key and requests go directly to the provider. There is no Local Dev Tools server involved.

Supported providers:

- **OpenAI** (GPT models)
- **Google Gemini**
- **LM Studio** (local models)

## Initial Setup

1. Add your API key(s) in Settings
2. Configure your persona description
3. Set your Twitter handle and LinkedIn URL
4. Select your preferred AI model

## How to Use

1. Select an AI model from the dropdown
2. Navigate to the date you want to generate content for
3. Click "Generate Content"
4. Review generated content cards
5. Use keyboard shortcuts to manage the queue:
    - **Arrow keys** - Keep or dismiss content
    - **C** - Copy content
    - **O** - Open the platform (Twitter/LinkedIn)

## Content Types

| Platform | Content Style                     |
| -------- | --------------------------------- |
| Twitter  | Short-form posts based on commits |
| LinkedIn | Professional updates and insights |
| Blog     | Long-form content ideas           |

## Local vs Online

- **Online indicator** - Using cloud AI (OpenAI, Gemini)
- **Local indicator** - Using LM Studio with local models

When using LM Studio, all processing happens on your machine and no data is sent externally.

## Persona

Configure your persona in Settings to customize the tone and style of generated content. This helps the AI understand your voice and target audience.
