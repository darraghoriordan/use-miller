Compare times across multiple timezones simultaneously. Perfect for scheduling meetings across global teams or checking business hours in different regions.

## Features

- **Multiple timezones** - Compare up to 8 timezones at once
- **Live updates** - Change time in one row, see all others update
- **Timezone search** - Find any timezone quickly
- **Persistent settings** - Your timezone selections are saved locally
- **Reset to now** - Quickly reset all times to the current moment

## How to Use

1. Click "+ Add Row" to add timezones (up to 8)
2. Select a timezone for each row
3. Adjust the time in any row to see conversions
4. Click "Reset to Now" to set all times to current time

## Use Cases

- **Meeting scheduling** - Find a time that works across offices
- **Business hours** - Check if a location is open
- **Travel planning** - Understand time differences at destinations
- **On-call handoffs** - Coordinate shifts across regions

## Tips

- Your selected timezones are saved and will persist between sessions
- Click the remove button on a row to delete it
- Use "Clear All" to start fresh with a new set of timezones
